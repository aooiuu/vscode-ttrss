## 功能

- 阅读状态标记 (根据文章、分类、列表一键标记)
- 分类、特殊分类、多级分类展示
- 文章搜索
- 阅读页面自定义 html 注入
- 无图模式
- 自定义读取规则 (view_mode: all_articles, unread, adaptive, marked, updated)
